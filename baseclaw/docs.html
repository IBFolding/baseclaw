Contract Templates

Pre-built smart contract templates you can deploy with one click:

🪙 ERC20 Token

Standard fungible token with mint/burn capabilities.

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.19;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";

contract MyToken is ERC20 {
    constructor() ERC20("MyToken", "MTK") {
        _mint(msg.sender, 1000000 * 10 ** decimals());
    }
}

🖼️ ERC721 NFT

Non-fungible token collection with metadata support.

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.19;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";

contract MyNFT is ERC721 {
    uint256 private _tokenIdCounter;
    
    constructor() ERC721("MyNFT", "NFT") {}
    
    function mint(address to) public returns (uint256) {
        uint256 tokenId = _tokenIdCounter++;
        _safeMint(to, tokenId);
        return tokenId;
    }
}

Security

🔐 Built-in Protections

  • Path traversal fix — Prevents directory escape attacks
  • Shell injection fix — Sanitizes all command inputs
  • 24h approval expiry — Admin actions time out
  • Wallet file permissions — 0o600 restricted access
  • Mainnet code-level block — Requires explicit approval
  • Social autopost block — No accidental tweets
  • Prompt injection filter — Blocks malicious inputs
  • Project count limit — Max 50 projects
  • Template overwrite guard — Prevents accidental overwrites

⚠️ Best Practices

  4. Start with small amounts
  5. Verify contracts on BaseScan

FAQ

❓ Do I need to know coding?

No! Just describe what you want in plain English. Agent BLUE handles all the Solidity code, testing, and deployment.

❓ What chains are supported?

Currently Base (Coinbase L2) with Sepolia testnet and mainnet. More chains coming: Optimism, Arbitrum, Polygon.

❓ Can I make money?

Yes! Create tokens, NFTs, paywalls, subscriptions, trading bots, DeFi yield strategies, and more.

❓ Is my wallet safe?

Agent BLUE never stores private keys. We use read-only wallet connections and require explicit approval for every transaction. Set spending limits in Wallet Policy skill.
