# Mobile Preview Skill

## Overview
Preview generated webpages on different device sizes before deployment.

## Device Sizes
- **Mobile Small** — 320px (iPhone SE)
- **Mobile** — 375px (iPhone 12/13/14)
- **Mobile Large** — 428px (iPhone 14 Pro Max)
- **Tablet** — 768px (iPad Mini)
- **Tablet Large** — 1024px (iPad Pro)
- **Desktop** — 1280px+

## Features
- **Device Frame** — Visual phone/tablet frame around preview
- **Rotate** — Toggle between portrait and landscape
- **Touch Simulation** — Simulate touch interactions with mouse
- **Network Throttling** — Simulate slow connections
- **Scroll Simulation** — Test scroll performance

## Usage
```javascript
baseclaw.preview.mobile()
// Opens mobile preview panel
```

## UI Integration
- Device selector dropdown
- Rotate button
- Touch mode toggle
- Network speed selector
