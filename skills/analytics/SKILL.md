# Analytics Dashboard Skill

## Overview
Track webpage performance and user interactions after deployment.

## Metrics
- **Page Views** — Total visits, unique visitors
- **Contract Interactions** — Connect wallet, transactions
- **Geography** — Visitor locations
- **Devices** — Mobile vs desktop usage
- **Referrers** — Where visitors come from
- **Conversion** — CTA clicks, form submissions

## Features
- **Real-time** — Live visitor count
- **Historical** — Daily/weekly/monthly charts
- **Export** — CSV, PDF reports
- **Alerts** — Notify on milestones

## Usage
```javascript
baseclaw.analytics.show()
// Opens analytics dashboard
```

## Integration
- Google Analytics
- Mixpanel
- Custom events from contract interactions
