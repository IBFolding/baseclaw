# Drag & Drop Builder Skill

## Overview
Customize the generated webpage by dragging and dropping components.

## Features
- **Drag Components** — Hero, features, stats, team, CTA, footer
- **Reorder Sections** — Move sections up/down
- **Edit Content** — Click to edit text, images, links
- **Add/Remove** — Add new sections or delete existing
- **Save Layout** — Persist custom layout to project

## Components
- **Hero** — Big headline + subtitle + CTA buttons
- **Features** — Grid of feature cards with icons
- **Stats** — Numbers/metrics display
- **Team** — Team member cards with photos
- **CTA** — Call-to-action section
- **Footer** — Links, copyright, social

## Usage
```javascript
baseclaw.builder.open()
// Opens drag-and-drop interface
```

## UI Integration
- Sidebar with draggable components
- Canvas area for dropping
- Properties panel for editing
- Preview button to see result
