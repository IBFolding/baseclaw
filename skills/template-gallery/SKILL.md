# Template Gallery Skill

## Overview
Visual gallery showing all available templates with live previews before selection.

## Features
- **Thumbnail Previews** — Screenshot of each template
- **Live Demo** — Interactive preview with mock data
- **Template Info** — Description, features, best use cases
- **Comparison View** — Side-by-side template comparison

## Templates

### ERC20 Token Launch
- **Preview**: `templates/erc20-page/preview.png`
- **Best For**: Token sales, ICOs, airdrops
- **Features**: Mint, transfer, balance, staking info

### NFT Collection
- **Preview**: `templates/nft-page/preview.png`
- **Best For**: Art collections, gaming NFTs, memberships
- **Features**: Gallery, mint, metadata, royalties

### Staking Pool
- **Preview**: `templates/staking-page/preview.png`
- **Best For**: Yield farming, liquidity mining
- **Features**: Stake/unstake, rewards, APY calculator

### DAO Dashboard
- **Preview**: `templates/dao-page/preview.png`
- **Best For**: Governance, voting, proposals
- **Features**: Proposals, voting, execution, treasury

## Usage
```javascript
baseclaw.gallery.show()
// Opens modal with all templates
```

## UI Integration
- Grid layout with hover effects
- Click to preview full-screen
- Select to generate project
