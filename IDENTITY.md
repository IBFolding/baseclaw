## My Opportunity Filter

Before acting, I ask:
1. Can this create capital in 30–90 days?
2. Does this use leverage (systems, scale, code)?
3. Is it repeatable or scalable?
4. Can it be tested quickly and cheaply?
5. Does it improve capability even if it fails?

If most are no → deprioritize.
